# 🎓 Портфоліо Three.js студента

Ласкаво просимо до мого портфоліо веб-розробки! Цей репозитарій містить мої проекти, пов'язані з WebGL та Three.js.

## 📁 Вміст

- **index.html** - Портфоліо-сторінка студента
- **scene.html** - Інтерактивна 3D сцена з текстурою відео на основі Three.js

## 🚀 Посилання

👉 **[Переглянути портфоліо на GitHub Pages](https://visitorxx.github.io/threejs-student/)**

- 📱 Повна адаптивність для мобільних пристроїв
- 🎬 Three.js сцена з анімованою текстурою відео
- ⚡ Інтерактивне керування мишею

## 📚 Навчання

Цей проект створений як частина навчання 3D веб-графіки та фундаментальних концепцій:
- Керування 3D сценою в браузері
- Текстури та матеріали
- Освітлення в 3D просторі
- Взаємодія з користувачем

## 💡 Як користуватися

1. Відкрийте [portfolio page](https://visitorxx.github.io/threejs-student/)
2. Натисніть на посилання "Three.js сцена з відео"
3. Взаємодійте зі сценою:
   - **Мишка** - обертання сцени
   - **Колесо мишки** - масштабування
   - **Кнопки** - контроль відео
