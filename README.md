## 🏃 Запуск локально

Рекомендується запускати через локальний HTTP сервер (браузери часто блокують автозапуск відео із локальних файлів).

Швидкий спосіб з Python (якщо встановлений):
```bash
# з кореня проекту
python -m http.server 8000
# відкрити http://localhost:8000/index.html
```

Або з Node (якщо встановлений npm):
```bash
npx http-server -c-1
```

Після запуску сервера відкрийте `http://localhost:PORT/index.html` і перейдіть до `scene.html`.

git push -u origin main

3. (Опціонально) Увімкніть GitHub Pages у налаштуваннях репозиторію: виберіть гілку `main` та папку `/ (root)` або `/docs`.
